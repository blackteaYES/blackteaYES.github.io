                                        # 1.手机拍摄及构图方法

                                        # 拍摄视频三大名词

                                          拍摄视频一定要知道的三个名词：像素、宽容度、焦距。

                                        # 什么是像素？

                                          视频中老师的解释：图片是由一个又一个的色彩小方格组成的，同样的画面大小里，小方格越多，像素就越高，看到的画面就越清晰。

                                          文心一言对像素的解释：像素是指在由一个数字序列表示的图像中的一个最小单位，称为像素。它是构成数码照片的基本单元，通常以像素每英寸PPI（Pixels Per Inch）为单位来衡量分辨率的高低。

                                          百度百科对于像素的解释：https://baike.baidu.com/item/像素?fromModule=lemma_search-boxopen in new window

                                        # 什么是宽容度？

                                          视频中老师的解释：不要同时把有过亮和过安的部分放在同一个画面里。

                                          文心一言的解释：宽容度是指胶片所能正确容纳的景物亮度反差的范围，能将亮度反差很大的景物正确记录下来的胶片称为宽容度大的胶片，反之则称为宽容度小的胶片。一般说来胶片的宽容度应该是越大越好。宽容度小的胶片，常会使景物明、暗部分在影像上得不到正确反映，损害影像的真实性。此外还有在使用上的曝光宽容度、显影宽容度等，都是指使用中的允许范围。

                                          百度百科对于宽容度的解释：https://baike.baidu.com/item/宽容度?fromModule=lemma_search-boxopen in new window

                                        # 什么是焦距？

                                          视频中老师的解释：一般手机都是等效24mm左右的广角镜头，广角也就是在同一位置相对标准镜头会摄入的景色范围更广，这在实际拍摄中也就是说手机拍摄的畸变，会比标准镜头的畸变更强。

                                          什么是广角镜头？

                                          https://baike.baidu.com/item/广角镜头?fromModule=lemma_search-boxopen in new window

                                          什么是摄影畸变？

                                          https://baike.baidu.com/item/畸变/3199484?fromModule=search-result_lemma-recommend#1open in new window

                                          文心一言的解释：焦距，是光学系统中衡量光的聚集或发散的度量方式，指平行光入射时从透镜光心到光聚集之焦点的距离。具有短焦距的光学系统比长焦距的光学系统有更佳聚集光的能力。简单的说焦距是焦点到面镜的中心点之间的距离。照相机中 焦距f<像距<2f 才能成像。

                                          百度百科对于宽容度的解释：https://baike.baidu.com/item/焦距/1880759?fromModule=lemma_search-box#3open in new window

                                        # 稳定器

                                        # 人肉稳定器

                                          上半身发力系统 -- 手   控制手臂与身体的位置，形成三角支架稳定结构。

                                          但是往往拍摄的时候需要移动，这个时候就需要下本身发力。

                                          下半身发力系统 -- 腿   通过半段的姿势和均匀的步子节奏，缓冲掉移动带来的上下抖动。

                                           腿微微弯曲，把自己下半身变成一个弹簧系统，平衡掉走动时带来的上下波动，加上现在手机自带的防抖效果，就可以拍摄出稳定的视频了。

                                        # 构图

                                          拍摄时总会思考一个问题，如何才能提升画面的美感呢？

                                          我们在选取摄影画面时，不要只考虑单一的元素，还要充分考虑前景、中景和背景与被摄主题的主从关系。

                                          比如将被摄主体置于镜头的前景，那背景丛深虚化或透过窗户，并且可以参考分割线来安排拍摄物体的位置。

                                        # 三分构图

                                          使用三分构图法最大的优势是使复杂的画面构成变得简单易操作，需要注意的是一定要认真找出画面中丛横排列的线条，线条越明显，越容易使用三分构图法，越容易拍出好的视频。

                                          所以当大脑一片空白时，首先要想到三分法则。

                                          此外由横向、纵向两个三分线构成的九宫格辅助线中的焦点也是非常有用的，特别时要推出主角的画面，当拍摄犹豫不定时，不知道怎么构图，可以尝试将主角放在焦点的位置，而可以直接辅助到三分法构图的方法，就是开启手机相机中的网格模式。

                                          在录像模式下，带着九宫格去观察屏幕内的画面。

                                          把线条的焦点对准你要拍摄的主体。

                                        # 对称构图

                                          对称构图：即按照一定的对称轴或对称中心，使画面中景物形成轴对称或者中心对称，常用于拍摄建筑、隧道等等，给观众以稳定、安逸、平衡的感觉，我个人就很喜欢把主人公放在画面中间，这样就很清楚的利用构图突出画面里的主角了。

                                        # 框架构图

                                          框架构图：将画面重点利用框架框起来的构图方式，会引导观者注意框内的景象，这种构图方式在短视频中会产生一种窥视的感觉，让画面充满神秘感，来引发观众的观看兴趣，实际拍摄中运用有难度，需要找到极其合适的景物。

                                        # 对角线构图

                                          对角线构图：多用于人物拍摄，是指主角处在画面对角线的位置，能表现出很强的动感、不稳定性或者生命力等感觉，给观众以更加饱满的视觉体验，使用此类构图法更多的是用来叙述环境，着重突出人物，我个人的短视频也常常用对角线构图来表现人物。

                                        # 尝试奇特的角度

                                          从初学者进阶为高手的一个重要前提，就是创意，使用正常角度拍摄，见到什么拍什么，几户谁都能做到，不过想要脱颖而出，就要尝试改变拍摄的角度，比如趴在草地上、站在楼顶或者梯子上，比较高或者比较低的视野进行拍摄，或者人为制造一个有趣的障碍物，总有一个能让你收获意外的惊喜。

                                        # 构图小办法

                                          多找角度，靠腿跑，多进行实践，实践出真知，笨办法有时候是最实用的。

                                        # 如何用光

                                          光线在摄像中是反映现实的必要条件和基础， 没有关线就没有影像，我们主要讲下如何利用自然光，复杂的打光对于小白的技术和成本要求太高，不太会在日常的拍摄中用到，我们最多用补光灯和反光板就算很厉害咯。

                                          自然光虽然采用方便，却是一种变化莫测、难以掌握的光线，其亮度和颜色都处在不断的变化当中，所以我们来到拍摄地点的第一个事情就是要辨认环境光的强弱和方向硬朗的直射光线。

                                        1. 硬朗的直射光
                                          顺光的优点：色彩还原好，保护度高，呈向清晰，低感光度下画面质感强，更接近其原型，环境细节丰富曝光比较容易把握。

                                          顺光的缺点：缺乏空间表现力，画面相对较平，且光质较硬，因此，选择顺光拍摄时要利用镜头的景深特点，突出人物，打破平淡。

                                          所以在拍摄时最好是顺光拍摄，自然光加补光拍摄，在正常光线不足的时候就需要使用反光板进行补光，补足视频主角所需要的亮度。

                                        2. 丰富多彩的斜射光
                                          傍晚的光线转向柔和，有人认为以日落时为最佳，在日落之前有很多很好的拍摄时机，当太阳消失在地平线上，夕阳的最后一抹余晖消失的时候，此时的天空呈深蓝色，而夕阳的光线是橙色。

                                          是我最喜欢的一组对比色，很容易出大片，但拍摄动作一定要快，因为这样的美景稍纵即逝。

                                          根据个人经验，最美的光线从出现到结束，一般只有半小时到一个小时左右。

                                          逆光、侧逆光的优点，空间立体感更强，影调和层面更丰富，整体环境的通透度高，人物会出现轮廓光，很容易营造出唯美的氛围。

                                          缺点是曝光和细节较难控制，曝光点的选择很重要，有时还需要人为补光，补光灯和反光板就在这个时候用，来控制画面的反差。

                                        # 常见的补光手动

                                          LED灯补光，用LED灯的时候，要注意不要让灯离被拍摄者很近，这样补出来的效果很差。

                                          光线的选择：

                                        1. 人为补光
                                          LED灯补光：使用补光灯最好距离是稍远。
                                          反光板补光：需要摄影助理协助。

                                        2. 自然光的运用（柔和散射光）
                                          太阳光线被云层或雾气所挡而形成的柔和、均匀的照射状态，如多云或阴雨天。

                                          柔和的散射光线，太阳光被云层或雾气所挡，而形成的柔和均匀的照射状态。

                                          拍摄时的注意事项：

                                          散射光的运用：
                                            多云天气：有利于表现立体感和质感
                                            阴雨天气：巧用地面积水，可以得到更好的效

                                          阴天条件下的注意事项：
                                            拍摄距离的远近与色彩的饱和度成正比，所以尽量选择鲜艳的元素作为前景，提高画面反差，避免影调平淡。

                                        # 不为人知的小技巧

                                        1. 手机的放大和缩小拍摄

                                          边拍边用放大功能进行视频拍摄会产生一种高级感，也可以加一些节奏感在视频素材里，但是手法忌讳犹豫，一定要果断，干净利落。

                                        2. 手动设定曝光与对焦

                                          按下屏幕的某个区域来进行自动对焦或者锁定对焦。

                                        3. 把拍到的视频混合

                                          把拍摄到的视频进行剪辑，提高观看的乐趣，但是需要注意的是，拍摄的视频角度不能都一样，要从多角度去拍，特写、中景、全景或者体罚重复拍摄同一运动的状态，从多方面去展现。

                                        4. 动态展现

                                          视频和照片最大的不同就是一个是动态展现，一个是静态展现。

                                          要拍摄出动感的视频就要注意拍摄的时间，每一个镜头只要是可以动起来都不要静止在哪里，哪怕是微微的推拉或者平移。

                                        5. 色彩应用

                                          灵活运用丰富美丽的色彩，像我个人偏爱对比色。

                                        6. 老师的温馨提示

                                          • 在拍摄之前建议将手机调为飞行模式，防止被电话、短信、微信等其它通知影响注意力等。
                                          • 保证手机有充足的电量。
